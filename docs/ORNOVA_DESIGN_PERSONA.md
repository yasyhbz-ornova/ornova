# ORNOVA Design Persona & Taste Profile

Codex must read this file before any UI, layout, copy, or visual task.

## Core Truth

Ornova is NOT handmade.

Yas designs jewelry digitally using:
- MatrixGold / Matrix
- ZBrush
- PC-based digital jewelry design workflow
- production-aware jewelry modeling
- technically buildable silver jewelry design

Forbidden terms:
- handmade
- handcrafted
- crafted by hand
- دست‌ساز
- دست ساز

Correct language:
- طراحی دیجیتال جواهر
- طراحی قابل ساخت
- طراحی شده با نگاه فنی و هنری
- طراحی اختصاصی جواهرات نقره
- طراحی با ابزارهایی مانند MatrixGold و ZBrush
- طراحی با workflow دیجیتال و قابل تولید

## Visual Taste

Ornova should feel:
- moonlit
- silver
- quiet
- feminine
- Persian
- intimate
- emotionally warm
- slightly melancholic
- poetic but professional
- minimal but not empty
- artistic but buildable

Preferred palette:
- Warm Ivory
- Moonlit Silver
- Deep Graphite
- Midnight Navy
- Deep Sapphire only as a small accent

Avoid:
- random colors
- harsh black
- gold-heavy palette
- childish contrast
- Divar/classified-ad feeling
- dry admin dashboard feeling
- crowded navigation
- too many boxes
- everything on one page

## Layout Taste

When unsure, choose:
- fewer items
- more spacing
- calmer rhythm
- stronger hierarchy
- more premium product presentation
- less text
- clearer page separation

Header must stay minimal.

Preferred main nav:
- خانه
- فروشگاه
- طراحی اختصاصی
- طراحی انگشتر
- راهنما

Secondary actions:
- ورود / حساب من
- سبد درخواست

Do not put every guide, FAQ, collection, or contact link in the top header.

## Home Page

Home should guide the user into three clear paths:
- فروشگاه
- طراحی اختصاصی
- طراحی انگشتر

Home may include:
- selected products preview
- short Ornova/Yas introduction
- guide entry

Home must not become:
- full dashboard
- full guide archive
- full builder
- full order form
- crowded landing page

## Product Pages

Product pages must feel like jewelry product sheets.

Use:
- product title
- short emotional description
- refined product details
- request price button
- order/request action
- question button

Do not use:
- fake prices
- تومان
- cheap marketplace layout
- dry table-like classified layout

Correct commerce language:
- درخواست قیمت
- افزودن به سبد درخواست
- ارسال درخواست
- قابل درخواست قیمت

## Ring Builder

Ring Builder must feel visual and premium.

It should include:
- پیش‌نمایش طرح شما
- visual option cards
- stone color swatches
- base shape thumbnails
- decorative detail thumbnails
- conceptual preview note
- ارسال طرح برای بررسی Ornova

Important note:
این پیش‌نمایش، طرح اولیه است. طراحی نهایی پس از بررسی Ornova و امکان‌سنجی فنی مشخص می‌شود.

## Custom Design

Custom Design page should focus on:
- استودیو طراحی اختصاصی Ornova
- ایده یا تصویر دارم
- مسیر سفارش اختصاصی

It may link to Ring Builder but must not mix the full Ring Builder into the same page.

## Copywriting

Persian copy should be:
- simple
- human
- emotional
- professional
- clear
- not too abstract
- not too stiff

Avoid phrases like:
- مالیخولیای خاموش
- رمانس پیچیده
- طنین‌انداز
- هنر ویراستاری

Use:
- آرام
- نزدیک به خاطره
- نقره‌ای
- ماهتابی
- شخصی
- ظریف
- قابل ساخت
- ماندگار
- طراحی دیجیتال جواهر
